# Calc

A simple web calculator with basic arithmetic: add, subtract, multiply, and divide.

## Run locally

```bash
npm install
npm run dev
```

Then open the URL Vite prints (usually `http://localhost:5173`).

## Build for production

```bash
npm run build
npm run preview
```

`npm run build` outputs static files to `dist/`, which you can host on any static web host.
